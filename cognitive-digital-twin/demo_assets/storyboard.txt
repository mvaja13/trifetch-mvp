PRODUCTION NOTES
===============================================================================

Recording Setup:
--------------
Screen Resolution: 1920x1080 (Full HD)
Frame Rate: 30 fps minimum
Recording Software:
  • macOS: QuickTime, ScreenFlow, or OBS
  • Windows: OBS, Camtasia, or ScreenFlow
Browser: Full screen mode (hide bookmarks bar)

Streamlit App Preparation:
-------------------------
1. Launch app: streamlit run src/app.py
2. Select a patient with clear decline (P005 recommended)
3. Test what-if sliders before recording
4. Set zoom level: 125-150% for readability
5. Hide unnecessary UI elements (sidebar toggle)

Audio Recording:
---------------
Microphone: Good quality USB or built-in (quiet environment)
Script: Write out voiceover word-for-word
Tone: Professional but approachable
Pacing: Moderate speed, clear enunciation
Recording: Record audio separately, sync in editing (recommended)

Editing Checklist:
----------------
☐ Trim dead space at start/end
☐ Add fade-in/fade-out transitions
☐ Sync voiceover precisely with visuals
☐ Add highlight boxes/arrows for key elements
☐ Ensure all text is readable
☐ Export at 1080p, 30fps
☐ File size < 100MB (compress if needed)

Optional Enhancements:
--------------------
• Background music (subtle, royalty-free)
• Animated transitions between scenes
• Lower-third graphics for statistics
• Mouse highlight effect for visibility
• Subtle zoom/pan for emphasis
